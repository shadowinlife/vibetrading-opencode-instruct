# 传统信号目录

## 概述

本文档列出 `traditional_signals.py` 支持的所有传统技术指标信号及其参数配置。

## 信号列表

| 信号 ID | 名称 | 类型 | 参数 | 说明 |
|---------|------|------|------|------|
| MA_CROSS | 均线交叉 | 趋势 | fast=5, slow=20 | 金叉买入，死叉卖出 |
| MACD | MACD信号 | 趋势 | fast=12, slow=26, signal=9 | MACD线上穿信号线买入 |
| RSI | 相对强弱 | 反转 | period=14, oversold=30, overbought=70 | 超卖买入，超买卖出 |
| BB | 布林带 | 波动 | period=20, std=2 | 下轨买入，上轨卖出 |
| ROC | 变化率 | 动量 | period=10 | 正值为多头信号 |

## 组合策略

- 投票法：多数信号一致时入场
- 加权法：按历史 IC 加权
- 门控法：主信号 + 确认信号
